# OKR Templates Library

## How to Use Templates

1. Choose a template that matches your goal
2. Fill in the placeholders `[...]`
3. Validate against Constitution rules (weights sum to 100%, outcomes not outputs)
4. Adjust ambition level (target 70% achievement)

## Template Customization Tips

1. **Adjust Ambition**: Start conservative if new to OKRs, increase 10% each quarter
2. **Industry Adapt**: B2B vs B2C metrics differ (B2C = 10x sample size, -20% rates)
3. **Team Size**: Scale targets proportionally to team capacity
4. **Timeframe**: Quarterly OKRs are most common, annual for strategic shifts
